Java的安装和环境配置

Java Downloads | Oracle

去Oracle官网，找到Java下载

选择Java8下载JDK8

x86表示适用于32位的电脑，x64适用于64位的电脑

需要先登录账号才能下载，如果没有账号就需要注册一个。

下载完之后双击安装

记住选择安装的路径

如果是默认安装路径，应该是C:\Program Files\Java

设置→

新建系统变量，变量名为JAVA_HOME，变量值为JDK的安装根目录

也就是这里

编辑PATH

编辑PATH变量，将刚刚新建的JAVA_HOME变量加上bin目录设置到PATH

或者是这种形式的

把%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;加到后面就行，注意不要漏掉分号。

win+R打开cmd

在命令行中分别输入java -version ,java,javac，若都出现下面，说明已安装成功。
